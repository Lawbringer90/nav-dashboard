🚀 开源一个自用的导航仪表盘 NavDashboard

之前一直用浏览器书签管理常用网站,但是太乱了找起来费劲。于是自己撸了一个导航站,分享给大家。

✨ 主要功能

📁 分类管理 支持自定义分类和图标
🔍 快速搜索 本地站点+多引擎搜索
🎨 毛玻璃 UI 颜值还不错
🔒 密码保护 后台管理有密码
📤 数据备份 一键导入导出
🧩 浏览器插件 右键快速添加网站

🌐 两种部署方式

Cloudflare 版
免费部署到 CF Pages
全球 CDN 加速
D1 数据库存储
GitHub Actions 自动部署

Docker 版
一行命令部署
数据本地存储
适合有服务器的朋友

docker run -d -p 3000:3000 ghcr.io/debbide/simple-nav-dashboard:latest

🧩 浏览器插件

这个是今天新加的功能
看到一个好网站 右键点一下就能添加到导航站
不用再手动复制粘贴了
